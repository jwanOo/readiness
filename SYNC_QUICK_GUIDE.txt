┌──────────────────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                                          │
└──────────────────────────────────────────────────────────────────────────────┘

   Problem: Button shows "Sync failed"
   ───────────────────────────────────
   Solution:
   1. Restart: npm run dev
   2. Check: curl http://localhost:5173/api/health
   3. Should return: {"status":"ok","syncAvailable":true}


   Problem: Sync takes too long (>60s)
   ───────────────────────────────────
   Solution:
   1. Check internet connection
   2. Check terminal for Python errors
   3. Verify Playwright installed:
      python3 -c "import playwright"


   Problem: No agents showing after sync
   ─────────────────────────────────────
   Solution:
   1. Verify CSV: grep -c "AI Agent" public/sap_ai_data.csv
      Expected: 32
   2. Hard reload browser: Cmd+Shift+R (Mac) / Ctrl+Shift+R (Win)
   3. Check filter is set to "AI Agent"

┌──────────────────────────────────────────────────────────────────────────────┐
│ 🔍 HOW IT WORKS                                                             │
└──────────────────────────────────────────────────────────────────────────────┘

   Click "Sync from SAP"
         ↓
   POST /api/sync
         ↓
   Vite Plugin intercepts
         ↓
   Spawns: node scripts/sync-sap-catalog.js
         ↓
   Runs: python3 Agents Discovery/downloader.py
         ↓
   Playwright downloads CSV from SAP
         ↓
   Copies to: public/sap_ai_data.csv
         ↓
   Frontend polls: GET /api/sync/status
         ↓
   Shows: "✅ 371 use cases loaded"
         ↓
   Reloads catalog data
         ↓
   ✅ 32 AI Agents ready!
